# How to run the IDS
Ensure to have installed the `uv` python package manager, more on it
[here](https://astral.sh/uv). Then clone the repository
```bash
https://github.com/danielepintore/network_security_project.git
```
Execute `run.sh`
```bash
./run.sh
```
